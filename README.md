# Showcase Projects

Collection de projets publics préparés pour un portfolio d'alternance en cycle ingénieur. Tous les projets utilisent des données fictives, des captures dédiées et une documentation orientée recruteur.

| Projet | Stack | Objectif |
| --- | --- | --- |
| [SenTechCare One](SenTechCare-One/) | Spring Boot, React, MySQL, JWT, PDF | Application métier complète : clients, abonnements, tickets, interventions, factures |
| [STC GEST 1](STC_GEST_1/) | React, Express, MySQL, PDFKit | Gestion catalogue IT, fournisseurs, sélections commerciales et exports PDF |
| [STC GEST 2](STC_GEST_2/) | React, Express, Prisma, MySQL, QR/barcode | Stock, mouvements, inventaires et étiquettes d'identification |

| [Symfony Business Manager](symfony-business-manager/) | Symfony, Doctrine, Twig, MySQL | Back-office métier avec KPI, clients, prestations et factures |
| [JEE Management App](jee-management-app/) | Java, Jakarta Servlet, JSP, Maven | Application Java entreprise MVC avec repository en mémoire |

## Qualité publication

- Données fictives uniquement.
- Captures intégrées dans chaque README.
- Fichiers `.env`, dépendances installées, builds et caches exclus via `.gitignore`.
- Secrets, emails personnels, numéros et credentials remplacés par des placeholders quand nécessaire.
- Les projets privés ou sensibles ne sont pas inclus.
- Les projets académiques gardent uniquement les sources, tests, assets et consignes utiles.
