# Power BI

Ce dossier est prévu pour le fichier Power BI du projet.

Le fichier `.pbix` n'est pas versionné par défaut dans ce projet, car il peut devenir lourd. Pour documenter le rendu :

1. créer le dashboard avec les fichiers du dossier `outputs/` ;
2. exporter une capture d'écran ;
3. placer la capture dans `screenshots/` ;
4. mettre à jour le README principal.

## Tables à importer

- `outputs/clean_members.csv`
- `outputs/kpi_summary.csv`
- `outputs/members_by_region.csv`
- `outputs/members_by_month.csv`
- `outputs/members_by_channel.csv`

## Visuels attendus

- Cartes KPI : total membres, actifs, taux d'activité, score moyen.
- Bar chart : membres par région.
- Courbe ou colonnes : inscriptions par mois.
- Donut ou bar chart : répartition par canal.
- Segmentations : statut, région, canal.
