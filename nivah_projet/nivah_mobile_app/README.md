# Nivah Mobile App

Application mobile Flutter/Dart pour une marketplace multi-boutiques.

## Fonctionnalités

- Authentification client et vérification email
- Catalogue produits, boutiques, catégories et marques
- Panier, commandes et suivi
- Stockage sécurisé des tokens
- Cache local avec SQLite
- Intégration paiement mobile money côté backend

## Stack

- Flutter / Dart
- Provider
- GoRouter
- Dio / HTTP
- sqflite
- flutter_secure_storage

## Lancement

```bash
flutter pub get
flutter run
```
