# Methodology

## Analyse effectuée

Le script suit une logique simple :

1. charger un fichier CSV ;
2. convertir les colonnes numériques ;
3. calculer un taux d'activité ;
4. agréger les données par segment ;
5. générer des insights textuels ;
6. produire un rapport Markdown.

## Pourquoi sans dépendances ?

Le projet est volontairement basé sur la bibliothèque standard Python pour être exécutable partout. Une version plus avancée pourrait ajouter :

- pandas ;
- scikit-learn ;
- clustering ;
- interface Streamlit ;
- LLM optionnel.

## Données

Les données sont fictives et anonymisées. Elles ne correspondent à aucun client ou projet réel.
