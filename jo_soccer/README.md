# JO Soccer

Projet web statique autour du tournoi olympique de football feminin.

## Objectif

L'application presente les selections nationales, les groupes, les stades et un parcours interactif de qualification. Elle met en pratique la manipulation du DOM en JavaScript et l'organisation d'une page web riche en donnees.

## Stack

- HTML
- CSS
- JavaScript vanilla
- Donnees locales en modules JS

## Fonctionnalites

- chargement dynamique des equipes dans les groupes ;
- affichage des informations d'une selection au survol ;
- tableau de medailles par pays ;
- donnees separees pour les equipes et les stades ;
- assets visuels pour les logos, stades et illustrations.

## Lancement

Ouvrir directement le fichier :

```text
jo_soccer.html
```

ou servir le dossier avec un serveur local :

```bash
python3 -m http.server 8080
```

## Note publication

Ce projet est un TP frontend. Il est conserve comme demonstration de JavaScript DOM, de structuration de donnees locales et d'integration HTML/CSS.
